 Student Performance Analysis using Python

📌 Description
This project is a simple Student Performance Analysis system built using Python.

It generates dummy student data and stores it in a CSV file. Then, it performs data analysis such as:
- Calculating average marks
- Finding the topper
- Department-wise analysis

The results are also visualized using a bar chart with matplotlib.

---

 Features
- Generate student dataset (CSV)
- Calculate overall average marks
- Identify topper
- Department-wise average calculation
- Data visualization using graphs

---

Technologies Used
- Python
- CSV module
- Matplotlib

---

 How to Run

1. Clone the repository or download files  
2. Open terminal in project folder  

Run the following commands:
python generate_data.py
python analysis.py
